16 июня 2026 года состоялся релиз бесплатного кроссплатформенного приложения для управления электронной почтой и новостными лентами Mozilla Thunderbird 152.0. Сборки проекта доступны для Windows, macOS и Linux в разделе Thunderbird Release в канале Release Channel. Это не ESR‑выпуск, если вам нужно решение с длительным сроком поддержки, то на сайте проекта доступен Thunderbird 140.11.1 ESR (или 115.18.0 для WIndows 7/8.1).
Выпуск Mozilla Thunderbird 140.0 ESR с долгосрочной поддержкой (Extended Support Release) произошёл в начале июля 2025 года. Обновления для этой версии будут выпускаться в течение года.
Основные изменения и доработки в Thunderbird 152:
включение устройств безопасности в корпоративных политиках;
настройка учётной записи Thundermail в один клик;
теперь можно использовать кнопку «Добавить» вместо «Новый» для создания учётных записей, календарей и адресных книг;
обновлена аутентификация OAuth GMail для использования PKCE;
при обнаружении адресных книг и календарей также проверяется имя хоста почтового сервера;
опция about:rights: заменена на local с хостовым url;
опция «Скрыть выполненные задачи» теперь также скрывает отменённые задачи;
исправлены 34 ошибки после выпуска предыдущей стабильной сборки, также в этой версии выпущены патчи безопасности против ранее обнаруженных уязвимостей.

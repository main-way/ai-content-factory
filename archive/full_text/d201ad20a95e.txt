Компания Microsoft выпустила PowerToys версии 0.100.0. Релиз содержит исправления и улучшения для нескольких модулей, а наиболее важные изменения касаются повышения производительности, уменьшения размера пакета, улучшений палитры команд, переработанного руководства по сочетаниям клавиш и Power Display.
Одно из наиболее заметных изменений в PowerToys 0.100.0 — это переход на .NET 10. Помимо модернизации кодовой базы, этот шаг значительно уменьшает общий размер приложения, а также, как утверждается, улучшает время запуска и общую отзывчивость. Для пользователей, которые постоянно запускают PowerToys в фоновом режиме, это даёт более плавную работу и меньшее потребление ресурсов в долгосрочной перспективе. Если предыдущий релиз 0.99.1 занимал 376 МБ, то последний — всего 272 МБ.
Microsoft также продолжает разработку Command Palette, и лаунчер получил новую утилиту под названием Extension Gallery. Она позволяет пользователям просматривать и устанавливать различные расширения, не выходя из Command Palette.
Также добавлено обновленное руководство по сочетаниям клавиш.
Учитывая, что Shortcut Guide был одной из первых функций PowerToys в эпоху открытого исходного кода, обновление приводит его в большее соответствие с современным языком дизайна, который Microsoft внедряет в рамках проекта.
Power Display же теперь позволяет пользователям управлять поддерживаемыми параметрами мониторов, такими как яркость, контрастность, громкость и цветовые профили, непосредственно из системного трея, не прибегая к физическим кнопкам монитора.
Инструмент Dock получил заметное обновление с поддержкой многомониторных конфигураций. Теперь пользователи могут настроить отдельный Dock для каждого дисплея, выбрать, на каких именно мониторах он должен отображаться, и решить, где будут показываться закреплённые команды.
Между тем расширение Performance Monitor теперь включает виджет батареи с подробной информацией о заряде и времени работы. Отдельные системные показатели, включая ЦП, память, графический процессор, сеть и батарею, также можно закрепить непосредственно на Dock для более быстрого доступа.
Несколько существующих модулей получили небольшие, но полезные улучшения. Microsoft продолжила доработку FancyZones, File Locksmith, Advanced Paste, Image Resizer, утилит для работы с мышью и других компонентов.
Наконец, релиз включает в себя исправления ошибок, направленные на повышение стабильности, снижение количества сбоев и устранение проблем, о которых сообщали пользователи.
Так, исправлено преобразование буфера обмена в JSON в расширенной вставке, теперь при ошибках чтения из буфера обмена возвращается пустой результат вместо появления исключения в #48124.
В апреле вышла версия PowerToys 0.99.0. В релиз вошли функция Grab And Move в FancyZones для управления окнами в стиле Linux, инструмент Power Display для настройки цветовых профилей и многое другое.

7 июня 2026 года состоялся релиз бесплатного с открытым исходным кодом мультиплатформенного редактора электронных книг Sigil 2.8.0, использующего Qt6 (и QtWebEngine). Проект предназначен для редактирования книг в формате ePub (как ePub 2, так и ePub 3).
Исходный код Sigil написан на С++ и Python и опубликован на GitHub под лицензией GNU General Public License v3.0. Сборки решения доступны для Windows (Windows 10 (1809) и новее), macOS (macOS 11 Big Sur и новее) и Linux, включая Flatpak.
В Sigil версии 2.8.0 исправлены 13 ранее найденных ошибок и багов после выпуска предыдущей версии, улучшена производительность, а также добавлены новые функции.
Основные изменения в Sigil 2.8.0:
отдельный диалог «Изменение порядка чтения» для тех, кто не может использовать перетаскивание;
создана зона для перетаскивания файлов в epub-файлы;
переработана опция ImageTab для возможности базовой настройки изображений: поворот вправо, поворот влево, обрезка, изменение размера;
предоставление пользователям большего контроля над функцией Prettify с помощью файла prettyprint.xml, созданного в настройках Sigil;
добавлена возможность распаковки нового встроенного CSS для работы с электронными книгами;
ImageView автоматически открывает изображения, подогнанные под размер окна;
распознаются больше основных типов носителей для image/jxl и image/avif из спецификации epub 3.4;
добавлена поддержка подсветки синтаксиса json в CodeView для поддержки спецификации epub 3.4;
добавлены новые пункты меню инструментов epb3 для безопасного добавления и удаления навигации из опции OPF spine.

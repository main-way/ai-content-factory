Epic Games выпустила Unreal Engine 5.8 — обновление игрового движка, в котором сделали акцент на создании больших открытых миров, ускорении пайплайнов анимации, улучшении рендеринга и интеграции AI-инструментов. UE 5.8 станет последним крупным обновлением ветки Unreal Engine 5. После этого разработчики Epic Games продолжат поддерживать движок исправлениями багов и патчами безопасности, а основной фокус сместят на Unreal Engine 6.
Что нового в Unreal Engine 5.8:
LLM можно подключать к Unreal Engine через MCP. В движок добавили экспериментальный MCP-плагин, который позволяет LLM подключаться к проектам и получать необходимый контекст. Плагин уже умеет работать с Blueprints, ассетами, уровнями, материалами, мешами и другими сущностями. Разработчики могут расширять возможность плагина под собственные задачи.
Новый Mesh Terrain для сложных открытых миров. Появилась система Mesh Terrain для создания ландшафта открытых миров: холмов, равнин, нависающих скал, пещер, арок, тоннелей, островов и других форм. Инструмент работает с недеструктивными модификаторами. Если разработчик передвигает элемент ландшафта, то система автоматически пересобирает окружение.
Улучшенный Procedural Vegetation Editor для работы с растительностью. С помощью инструмента теперь можно создавать деревья, готовые к использованию с Nanite. Кроме того, растениям можно задать биологически правдоподобное поведение, чтобы они конкурировали за источники света, адаптировались к внешней геометрии и формировали кластеры.
Больше возможностей в MetaHuman. Система MetaHuman Collections теперь позволяет создавать сотни цифровых персонажей для мобильных устройств и тысячи для более производительных платформ. MetaHuman Animator также получил поддержку markerless motion capture для всего тела. Для захвата движения теперь не обязательно использовать костюмы, маркеры или шлемы с камерами. Достаточно всего одной внешней камеры.
MegaLights и облегчённый режим Limen. Инструмент MegaLights получил статус Production-Ready. Эта технология позволяет использовать большое количество динамических областей света с тенями, снижая шум и повышая качество картинки. Epic также заявляет об улучшении производительности до 60 fps на консолях текущего поколения. Появился и Lumen Lite — облегчённый режим глобального освещения. Он сохраняет визуальные эффекты классического Lumen, но требует меньше GPU-ресурсов.
Обновление для мобильной разработки. Unreal Engine Remote теперь позволяет тестировать мобильный ввод, включая тач-жесты, без сборки и деплоя на физическое устройство.
Обновление уже доступно на сайте Unreal Engine.

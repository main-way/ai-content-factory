Собрали ключевые обновления MWS Cloud Platform за последнее время. В этом выпуске — быстрые временные диски для ВМ, мостик между UI и IaC, улучшения в Managed PostgreSQL и новая большая языковая модель в каталоге, а ещё анонс вебинара.
Инфраструктура и сеть
В Compute появилась поддержка локальных дисков — временных NVMe-хранилищ, которые работают быстрее сетевых, но не гарантируют сохранность данных. При остановке ВМ данные стираются: после перезапуска машина может оказаться на другом хосте.
Где это может пригодиться:
— Кеши и буферы БД: в связке с сетевыми дисками или в кластерной схеме, чтобы потеря локального диска не означала потерю данных.
— CI/CD-сборки: ускорение pipeline за счёт локального хранения кешей зависимостей и промежуточных артефактов. Финальные результаты сохраняются во внешнее хранилище.
— ETL, batch-задачи, рендеринг: данные загружаются из постоянного хранилища, быстро обрабатываются локально, результат пишется обратно. Фича находится в бета-тестировании, и доступ к ней бесплатный. Чтобы получить доступ, напишите в поддержку через виджет в консоли.
CLI-эквивалент из веб-консоли
При создании виртуальной машины в консоли теперь можно нажать кнопку «CLI/Terraform» и получить готовую команду для CLI, повторяющую текущее действие.
Полезно для тех, кто переходит от ручного управления к автоматизации через Infrastructure as Code. Пока поддерживается только CLI для создания ВМ в Compute. Поддержка Terraform — в планах.
Платформа данных
Managed PostgreSQL: расширенные настройки и единый read-only endpoint
В консоли появились дополнительные параметры PostgreSQL, которые можно задать при создании кластера или изменить на лету: тонкая настройка autovacuum, планировщика запросов, транзакций и фоновых процессов под вашу нагрузку.
Также упростили работу с репликами на чтение — теперь доступен единый endpoint для read-only запросов. Больше не нужно вручную управлять подключениями к нескольким репликам. Удобно для отчётности, BI-систем и read-heavy сервисов.
ИИ-платформа
GPT Model Hub: модель gpt-oss-120b
В каталоге GPT Model Hub появилась текстовая модель gpt-oss-120b от OpenAI с 120 миллиардами параметров. Подходит для чат-ассистентов, обработки документов, суммаризации, генерации текстов и интеграций через API. Модель уже доступна в GA.

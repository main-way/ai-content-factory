Microsoft в ответ на негативные отзывы пользователей позволит им полностью отключить Bing в поиске Windows 11. Также компания заявила, что поиск будет отдавать приоритет локальным, а не веб-результатам, даже если ввести всего два символа.
Изменения появились в предварительных сборках. Microsoft продемонстрировала их на закрытой встрече участников программы Windows Insider.
В Windows Latest подтвердили, что уже ведётся работа над локальным поиском Windows. Microsoft тестирует новый переключатель, позволяющий отключать веб-поиск, чтобы инструмент работал полностью локально. Также можно будет деактивировать результаты поиска в Microsoft Store.
Сейчас, если искать в Windows Search приложение, которое не установлено, но есть в магазине, то появится список с кнопкой «Получить». Он исчезнет после того, как пользователь вручную отключит результаты поиска в Microsoft Store в поиске Windows. Эти изменения начнут внедряться для тестировщиков через несколько недель.
В настоящее время клиенты не могут отключить веб-интеграцию в поиске Windows. Единственный простой способ сделать это — изменить реестр Windows. Для этого требуется создать несколько записей в HKEY_CURRENT_USER > Software > Policies > Microsoft > Windows, чтобы заблокировать веб-интеграцию.
Microsoft тестирует ряд других улучшений для Windows Search, в том числе:
локальный поиск без Copilot, отображение результатов поиска в Microsoft Store, MSN, Bing и Rewards;
более быстрый поиск Windows, особенно если файлы содержат сложные имена;
приоритет локальным результатам даже при активации веб-поиска, когда пользователь вводит всего два символа.
Ранее в Windows Latest обнаружили новую запись в групповой политике Windows 11, которая не только отключает Copilot, но и навсегда удаляет ИИ-помощника с ПК. Вероятно, её добавили в апрельском обновлении.

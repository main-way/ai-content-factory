Новый медиаплеер Microsoft для Windows 11 раскритиковали за то, что он использует гораздо больше памяти, чем классический Windows Media Player. Кроме того, он скрывает некоторые популярные кодеки за платным доступом.
Сообщается, что современный медиаплеер использует около 377 МБ оперативной памяти в режиме ожидания по сравнению с примерно 103 МБ у старого — это примерно в 3,5 раза больше. Программа дольше открывает локальные видеофайлы, время запуска увеличивается примерно на 50% — с двух секунд у старого плеера до трёх секунд у нового.
Проблемы не ограничиваются производительностью. Microsoft теперь ограничивает воспроизведение HEVC (H.265) через платное приложение «HEVC Video Extensions» в Microsoft Store. Microsoft также подтвердила, что в Windows 11 версии 24H2 удален встроенный кодек AC-3 (Dolby Digital), а это значит, что новый медиаплеер на этих системах не может воспроизводить аудиодорожки AC-3 в нативном режиме.
Новое программное обеспечение заменяет Groove Music и классический Windows Media Player на всех компьютерах с Windows 11. Microsoft по-прежнему предлагает классический Windows Media Player в качестве дополнительного компонента, но явно продвигает новый Media Player как выбор по умолчанию для большинства пользователей. Зимой 2026 года Windows Media Player прекратил предоставлять метаданные для отображения в ОС. В Microsoft признали, что серверы этого сервиса отключены, а пользователям следует найти альтернативное решение от сторонних разработчиков.
Тем, кто не против использовать сторонние программы, можно рассмотреть такие плееры, как VLC, которые поставляются со своими собственными кодеками.
В июне состоялся релиз открытого медиаплеера Media Player Classic — Home Cinema 2.7.3 (MPC‑HC). Бесплатный видео‑ и аудиоплеер с открытым исходным кодом для Windows MPC‑HC основан на оригинальном проекте Guliverkli. Как продолжение оригинального Media Player Classic, MPC‑HC имеет простой интерфейс, а также работает практически с любым форматом медиа.
